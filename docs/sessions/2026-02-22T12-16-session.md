- **Endpoints**: `/api/auth/token`, `/api/auth/dev-token`, `/api/transactions`, `/api/portfolios`
- **Auth**: JWT stub with dev token; `get_current_user` dependency returns 401 on invalid token
- **Alembic**: Async template configured, reads `DATABASE_URL` from env

### Frontend (`web/`)
- **Stack**: Angular 19 standalone, Angular Material, SCSS
- **Structure**: `core/{services,guards,interceptors}`, `features/{dashboard,transactions,portfolios}`, `shared/models`
- **Components**: AppComponent (sidenav + toolbar), DashboardComponent, TransactionsComponent, PortfoliosComponent
- **Services**: ApiService, authGuard, authInterceptor

### Docker
- **Services**: `db` (postgres:17-alpine), `api` (FastAPI dev), `web` (Angular dev)
- **Ports**: 5432 (db), 8000 (api), 4200 (web)
- **Volumes**: postgres_data, bind mounts for hot reload

## Key Decisions
- Monorepo with `/api` and `/web` folders
- `uv` for Python package management
- Angular 19 standalone (no NgModules)
- JWT stub only (no login UI yet)
- asyncpg over psycopg2
- Migrations inside `api/alembic/`
